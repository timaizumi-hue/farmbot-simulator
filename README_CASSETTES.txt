FarmBot 練習シミュレーター v25.0 カセット式モード

概要:
練習モードA/Bを、本体に直接混ぜるのではなく modules/ 配下のモードパックとして整理しました。

使い方:
1. ホームを開く
2. 練習モードA または 練習モードB を押す
3. 目次画面から内容を選択
4. 選択内容を読み込んだ状態で本体モードが開始されます

練習モードA:
- 基本操作
- 水やり
- シークエンス
- 課題モード

練習モードB:
- 春野菜
- 夏野菜
- 冬野菜

注意:
今回の版では、練習A/Bの「入口と構造」を整理しました。
採点・3か月育成エンジンは次段階で modules/ 内へ追加していく想定です。

v25.1 追加:
- 練習モードA > 基本操作 を、目次だけでなく実際に進行するステップ式レッスンにしました。
- 手順: 畑マップで座標選択 → Move To → Home → Safe Z
- ユーザーが実際に操作すると、自動で次のステップへ進みます。
- 水やり / シークエンス / 課題モードは次段階で同じ仕組みに接続します。

=== v25.2 練習モードA 基本操作の固定課題化 ===
- 練習モードA > 基本操作を開くたびに、位置・水分・経路・植物配置を初期化します。
- 目標座標は X900 / Y450 / Z0 です。
- 畑マップ上に黄色の目標マーカーを表示します。
- 目標以外を選択してもレッスンは進みません。
- Move も目標付近へ到達した場合のみ次へ進みます。

=== v25.3 ===
練習モードAに「水やり」「シークエンス」「課題モード簡易版」を追加しました。
各練習は、開始時に位置・植物・水分・シークエンスを初期化します。
ユーザーが自由に触るだけではなく、黄色の目標位置や目標株に従って進む固定課題型です。
また、PC表示で動作レビューが大きくなりすぎないよう、右側の上下配分を少し畑マップ寄りに調整しました。


=== v25.4 ===
シークエンスレッスンの工程数を増やしました。
新しい流れ:
1. 目標株を選択
2. 目標へMoveを追加
3. 水量/半径を追加
4. 待機を追加
5. メッセージを追加
6. X0/Y0/Z0へ戻るMoveを追加
7. Run Sequenceで実行

Move追加では、目標MoveとHome位置Moveを簡易チェックします。


=== v25.5 ===
課題モードをシークエンス提出型へ変更。
ランダム課題株を生成し、目標半径・水量・待機時間を提示。
Run Sequence後に、場所・水量/半径・待機時間・工程順・帰還・効率を SS〜C で評価します。
