FarmBot 練習モードB 育成モード v25.7

この版では、練習モードBを「育成ゲーム」として一貫するよう整理しています。

主な仕様:
- 春野菜 / 夏野菜 / 冬野菜を新規開始できます。
- 植物は新規開始時に自動生成され、途中で追加・削除・種類変更はできません。
- 保存データ読込から途中再開できます。
- 3か月(90日)を約1時間で進行します。
- 停止 / 進む / 早送り / 1日進める を使用できます。
- 天気、気温、湿度、雨量は緩やかに変動します。
- 水分、健康、成長、収穫スコアが時間で変化します。
- 予約シークエンスで「何日ごとに水やり」を設定できます。

注意:
- このモードは栽培の流れを学ぶための教育用です。
- 天気は極端な失敗要因ではなく、管理判断の補助として扱います。
