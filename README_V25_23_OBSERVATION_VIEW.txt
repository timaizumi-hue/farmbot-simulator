FarmBot v25.23 観察ビュー化

目的
- 動作レビューと手元カメラの役割重複を減らす
- 手元カメラを「植物の状態を観察するビュー」に変更
- 静止画・写真風ではなく、葉パーツを組み合わせるアニメーション表現へ寄せる

変更点
- 手元カメラ表記を観察ビューに変更
- 観察ビュー内の植物描画を画像スプライト依存ではなく、葉パーツのベクター描画に変更
- 水分状態に応じて葉の色・縮み・垂れ方が変わるようにした
- 適正時は広がり、水不足時は小さく垂れ、過湿時は暗く重い表現に寄せた
- FarmBot全体の動きは従来の動作ビュー側に残した

触っていないこと
- 練習モードBの進行
- ホーム復帰・再開処理
- 時間進行・早送り
- 水分ロジック本体
- 畑マップ構造
