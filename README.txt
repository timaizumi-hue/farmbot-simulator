FarmBot 練習シミュレーター v20.7
分割構成・手元カメラ情報強化版

起動方法
1. OPEN_HTML.bat を開く
2. うまくいかない場合は index.html を直接開く

今回の主な変更
- CSS と JavaScript を分割構成へ移行
- 手元カメラを拡大し、文字の押し込みを減らして情報欄を別配置
- 植物に必要な水分量・現在水分量・葉の受水率・根元状態・環境条件を整理して表示
- 散水半径を広げた時、手元カメラの散水範囲も広がるよう調整
- 葉の描写にランダム性・重なり・上下感・水滴表現を追加

注意
- Python は不要です
- USB からの直接起動を想定しています
