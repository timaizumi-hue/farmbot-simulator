FarmBot v25.9 練習モードB 育成HUD並行運用版

変更点:
- 育成モードBを本体画面の置き換えではなく、通常画面上のHUDとして表示します。
- 元のフリーモードのMove、周辺機器、水やり、シークエンス、畑マップ、動画レビューを使いながら育成できます。
- 育成開始時に植物を自動生成し、本体側の植物配置にも同期します。
- 植物の追加・削除・種類変更は育成モード中の前提から外しています。
- 本体のWATER OFF時に近くの育成植物へ水やり結果を反映します。
- 育成HUDは展開/最小化/保存/閉じるができます。

確認手順:
1. ホームから練習モードBを選ぶ。
2. 春/夏/冬野菜または読込を選ぶ。
3. 元の操作画面が表示されたまま、右下に育成HUDが出る。
4. 元のMove/Water/Sequenceを使って植物を管理する。
